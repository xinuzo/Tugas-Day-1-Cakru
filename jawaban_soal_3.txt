Fork adalah tindakan menyalin repositori proyek orang lain ke akun GitHub pengguna itu sendiri. Fork memberikan  salinan independen dari repositori tersebut yang bisa dimodifikasi tanpa memengaruhi repositori asli. Forking umumnya digunakan ketika seseorang ingin berkontribusi pada proyek open source atau ingin mengembangkan proyek yang sudah ada sesuai kebutuhan. Sedangkan, Pull request adalah permintaan untuk memasukkan perubahan yang telah dibuat di fork ke repositori asli.

Intinya, fork memungkinkan pengguna untuk memiliki salinan bebas dari repositori sumber, sementara pull request memungkinkan pengguna untuk mengajukan perubahan yang telah Anda buat untuk dimasukkan kembali ke repositori aslinya. Proses ini memfasilitasi kolaborasi dan kontribusi pada proyek open source di GitHub.
